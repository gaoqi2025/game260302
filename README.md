# 小猫跑酷游戏

一款科幻风格的小猫跑酷游戏，基于HTML5 Canvas开发。

## 游戏特色

- 科幻风格的视觉效果
- 可爱的霓虹小猫角色
- 外星机器人障碍物
- 流畅的跳跃体验
- 得分系统
- 开始界面和游戏结束界面

## 如何运行

### 本地运行

1. 克隆仓库
2. 进入项目目录
3. 运行以下命令启动本地服务器：
   ```bash
   python -m http.server 8000
   ```
4. 在浏览器中访问 `http://localhost:8000`

### 游戏操作

- 按空格键开始游戏
- 再次按空格键控制小猫跳跃
- 躲避外星机器人障碍物
- 只有跳跃躲避障碍物才能得分

## 技术栈

- HTML5 Canvas
- JavaScript
- CSS3

## 部署

项目已配置为可在Vercel上部署。只需将代码推送到GitHub仓库，然后在Vercel上导入该仓库即可。
